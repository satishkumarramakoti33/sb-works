SB Works is a full-stack freelancing platform where clients can post projects and freelancers can apply, get assigned, and complete projects.

---

## Tech Stack

Frontend:
- React.js
- Bootstrap
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

---

## Features

Client:
- Register and login
- Post projects
- View applicants
- Assign freelancer
- Delete projects

Freelancer:
- Register and login
- Browse projects
- Apply for projects
- View assigned jobs
- Mark jobs as completed

Security:
- JWT Authentication
- Role-based access control
